# Day 5 Schedule: Communication, Polish, And Project Studio

Assumed class block: 3 hours.

## Main Goal

Students revise exploratory plots into clearer communication figures and leave with concrete final project progress.

## Schedule

| Time | Segment | Materials | Notes |
|---:|---|---|---|
| 0:00-0:10 | Warm-up and project check-in | Verbal | Ask each student to name dataset/question status. |
| 0:10-0:25 | Exploration versus communication | `slides/day-05-communication-and-studio.qmd` | Set revision mindset. |
| 0:25-0:45 | Effective and honest scales | `modules/09_communication-polish/01_effective-and-honest-scales.qmd` | Discuss scale tradeoffs. |
| 0:45-1:05 | Color, accessibility, and highlighting | `modules/09_communication-polish/02_color-accessibility-and-highlighting.qmd` | Highlighting over rainbow defaults. |
| 1:05-1:20 | Saving visualizations | `modules/09_communication-polish/03_saving-visualizations.qmd` | Practical export workflow. |
| 1:20-1:30 | Break |  | Transition to studio. |
| 1:30-1:45 | Critique demo | Day 5 slides, one volunteer/example plot | Use feedback prompts. |
| 1:45-2:40 | Final project studio | Student project notebooks | Instructor circulates for support. |
| 2:40-2:55 | Pair feedback | Student plots/questions | One plot, one question, one next step. |
| 2:55-3:00 | Final wrap | Verbal | Clarify deliverables and deadlines. |

## If Time Runs Short

Cut the critique demo and go straight to project studio. Do not cut studio time below 45 minutes.

## Instructor Priorities

- Students identify a final project question.
- Students revise at least one figure for communication.
- Students write interpretation plus limitation.
- Students leave knowing their next concrete step.
