# Day 4 Schedule: Group Comparison And Association

Assumed class block: 3 hours.

## Main Goal

Students can compare groups, plot differences directly, and describe association between two numeric variables without overclaiming.

## Schedule

## If Time Remains

Use `practice/04-comparison-and-association/01_grouped-distributions-practice.qmd` if students need more time comparing distributions before summaries.

Use `modules/06_association/02_model-output-plots.qmd` as a short preview. Keep it conceptual: plot data first, then show fitted model lines by group.

Use `practice/04-comparison-and-association/05_model-output-plots-practice.qmd` only as optional extension practice.

## Instructor Priorities

- Students do not default to bar charts for every comparison.
- Students distinguish distributions, summaries, and differences.
- Students label difference direction clearly.
- Students describe association using direction, form, strength, spread, and unusual values.
- Students use association language, not causal language.
