# Data Manifest

Source: generated by `scripts/build-data-manifest.R`.

Unit of observation: one CSV file in the course data inventory.

## Variables

- `dataset_id`: filename without `.csv`.
- `dataset_kind`: `real`, `simulated`, or `metadata`.
- `file_path`: path to the CSV file.
- `codebook_path`: path to the matching codebook.
- `n_rows`: number of rows in the CSV.
- `n_columns`: number of columns in the CSV.
- `analog_pair_id`: identifier linking real datasets to their simulated analogs; missing for metadata files.
- `analog_topic`: short description of the pairing or metadata file.
- `paired_dataset_path`: path to the paired real or simulated file; missing for metadata files.
