# State Population Ranks Codebook

File: `data/real/state_population_ranks.csv`

Source: Census 2020 state population data from previous course materials, with corrected population values for states whose source rows were shifted.

Rows: 51

Columns: 4

## Unit Of Observation

Each row is one state or District of Columbia.

## Variables

- `state`: State abbreviation.
- `state_name`: Full state name.
- `population`: State population.
- `rank`: Population rank.

## Teaching Notes

- This dataset is good for simple rank examples and joins.
- It is best used with another state-level dataset rather than alone.
