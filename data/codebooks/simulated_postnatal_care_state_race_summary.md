# Simulated Postnatal Care State Race Summary

Simulated teaching dataset summarized from the summer 2025 spatial visualization example.

Each row is one state and one group.

## Variables

- `state`: state abbreviation
- `state_name`: state name
- `race_ethnicity`: compact group code
- `group`: display label
- `n`: number of simulated observations in the state/group
- `proportion`: proportion receiving comprehensive postnatal care

The `Overall` rows summarize all simulated observations in a state.
