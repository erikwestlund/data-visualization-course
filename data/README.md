# Data Directory

This directory contains the course datasets, simulated analogs, codebooks, and dataset manifest.

## Layout

- `real/`: cleaned real public teaching datasets.
- `real/wide/`: derived wide versions of selected real datasets.
- `simulated/`: simulated analog datasets with different topics but matching column names to real datasets.
- `codebooks/`: codebooks for every CSV listed in the manifest.
- `manifest.csv`: inventory of all real, simulated, and metadata CSV files.

## Manifest

Use `manifest.csv` to find datasets, codebooks, and simulated analogs.

Key columns:

- `dataset_id`: filename without `.csv`.
- `dataset_kind`: `real`, `simulated`, or `metadata`.
- `file_path`: path to the CSV file.
- `codebook_path`: matching codebook.
- `n_rows` and `n_columns`: dataset dimensions.
- `analog_pair_id`: shared ID for real/simulated pairs.
- `paired_dataset_path`: paired real or simulated file.

## Swapping Real And Simulated Data

Real and simulated paired datasets have matching column names. Use `manifest.csv` or `simulated/simulated_dataset_crosswalk.csv` to find the pair.

Example:

```r
readr::read_csv(here::here("data", "real", "nhanes_2021_2023_analysis.csv"))
readr::read_csv(here::here("data", "simulated", "neighborhood_health_survey_analysis.csv"))
```

## Rebuilding

- Real data prep scripts live in `scripts/prepare-*.R`.
- Simulated analogs are generated by `scripts/simulate-teaching-data.R`.
- The manifest is generated by `scripts/build-data-manifest.R`.
